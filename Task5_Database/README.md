# База данных SportClub

## Схема
- Members – участники
- Trainers – тренеры
- Workouts – виды тренировок
- MemberWorkout – связь N:N (какие тренировки посещает участник)
- Payments – платежи (1:N)

## Уникальный индекс
Создан на поле Email в таблице Members.
**Почему уместен:** часто используется для поиска при логине, гарантирует уникальность.
**Минусы:** замедление INSERT/UPDATE, дополнительное место.

## Как развернуть
1. Создать БД `SportClub` в pgAdmin.
2. Выполнить `01_create_tables.sql`
3. Выполнить `02_insert_data.sql`
4. Выполнить `03_queries.sql` для проверки.
